# Chonus

## Como executar o projeto

### Instale as dependencias do FRONTEND  e execute-o no modo de desenvolvimento
```
npm i
```
```
npm run serve
```

### Instale as dependencias do BACKEND DE TESTE
```
cd /backend
```
```
npm i
```
```
npm start
```

### URL do backend
```
http://localhost:3003/
```
### URL do frontend
```
http://localhost:8080/
```
